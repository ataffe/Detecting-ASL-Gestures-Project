The Python code was written in Python 2.7; there is no guarantee other versions of Python will run successfully.
The Python code makes use of numpy and must be installed.
The Python code makes use of Scikit Learns Python library and must be installed. There is instructions of how to do so on their website.

To run the Python code, navigate to a folder labeled with a persons name and open a terminal. The kNN.py file is the main Python file.

In the terminal type: python kNN.py

The code will begin to run and upon terminate a csv file by the name k_scores.csv will have been created. This file will hold the accuracy of the
tests peformed by the Python code.
